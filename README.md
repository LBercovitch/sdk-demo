# sdk-demo
A simple application using ArcGIS Maps SDK.

This app uses [react](https://react.dev/), [vite](https://vite.dev/), and [tailwind](https://tailwindcss.com/).

Setup:

- Clone the repo to your local dev environment.
- Navigate to the project's folder with `cd sdk-demo`.
- Run `npm i` to install all node packages.
- Run `npm run dev` to start the server.
- Copy the url from the server output. It should be `http://localhost:5173/`.

Contribution:
